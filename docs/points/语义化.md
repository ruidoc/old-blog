# 语义化

什么是语义化？就是用合理、正确的标签来展示内容，比如 h1~h6 定义标题。

好多同学的 HTML 标签是清一色的 div，这就是典型的不遵循语义化。

语义化优点：

- 语义化更具可读性
- 语义化有利于SEO

![html5-768x508](/assets/html5-768x508.png)

### 列举语义化标签

**1、\<header>**

定义网页的头部区域。

**2、\<nav>**

定义网页的导航区域。

**3、\<main>**

除`<header>，<nav>，<footer>`外的主要内容区域。

**4、\<article>**

定义网页的文章段落区域。

**5、\<aside>**

定义网页的侧边栏区域。

**6、\<footer>**

定义网页的底部区域。

**7、\<section>**

定义网页主区域下的子区域。
